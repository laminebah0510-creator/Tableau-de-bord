# Tableau de Bord Interactif des Ventes – Power BI

## Présentation du projet

Ce projet consiste en la conception et le développement d'un tableau de bord interactif à l'aide de Microsoft Power BI pour une entreprise spécialisée dans la vente de vêtements. À partir de données commerciales stockées dans plusieurs fichiers Excel et csv, l'objectif était de transformer des données brutes en informations stratégiques permettant d'améliorer le suivi des performances et la prise de décision.

Le tableau de bord offre une vision globale de l'activité commerciale grâce à des indicateurs clés de performance (KPI), des visualisations dynamiques et des filtres interactifs facilitant l'exploration des données.

---

## Objectifs

Les principaux objectifs de ce projet sont :

* Analyser les performances commerciales de l'entreprise ;
* Suivre l'évolution du chiffre d'affaires et des ventes ;
* Identifier les produits et catégories les plus performants ;
* Mettre en évidence les tendances de consommation ;
* Fournir un outil d'aide à la décision aux responsables commerciaux.

---

## Technologies utilisées

| Outil                           | Utilisation                             |
| ------------------------------- | --------------------------------------- |
| Microsoft Power BI              | Création du tableau de bord             |
| Microsoft Excel                 | Source des données                      |
| Power Query                     | Nettoyage et transformation des données |
| DAX (Data Analysis Expressions) | Création des mesures et indicateurs     |

---

## Source des données

Les données exploitées dans ce projet proviennent de fichiers Excel contenant des informations relatives :

* aux ventes réalisées ;
* aux produits commercialisés ;
* aux clients ;
* aux dates de transaction ;
* aux catégories de vêtements.

Avant leur intégration dans Power BI, les données ont été nettoyées, transformées et structurées afin de garantir leur qualité et leur cohérence.

---

## Fonctionnalités du tableau de bord

### Analyse des ventes

* Suivi du volume des ventes ;
* Analyse de l'évolution des ventes dans le temps ;
* Comparaison des performances entre différentes périodes.

### Analyse du chiffre d'affaires

* Chiffre d'affaires total ;
* Chiffre d'affaires par catégorie ;
* Répartition des revenus selon les produits.

### Analyse des produits

* Identification des meilleures ventes ;
* Classement des produits les plus performants ;
* Analyse des catégories de vêtements.

### Visualisations interactives

* Graphiques dynamiques ;
* Cartes d'indicateurs (KPI) ;
* Segments et filtres interactifs ;
* Navigation intuitive entre les différents indicateurs.

## Méthodologie

Le projet a été réalisé selon les étapes suivantes :

1. Collecte et compréhension des données.
2. Nettoyage et transformation des données avec Power Query.
3. Modélisation des tables et création des relations.
4. Conception des mesures et indicateurs à l'aide de DAX.
5. Création des visualisations interactives.
6. Validation des résultats obtenus.
7. Optimisation de l'expérience utilisateur.
